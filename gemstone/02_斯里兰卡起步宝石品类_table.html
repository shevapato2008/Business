天然锆石
Zircon
（#3 内容爆款担当）		 ZrSiO4 中（重点关注）：可能天然含微量 U/Th，个别货“可测到”轻微放射性；建议来货用简易辐射仪抽检，异常批次淘汰 火彩强、视频效果好；香槟/金/蓝可系列化；适合做火彩教育（注意与CZ区分） 刻面圆/椭圆：
0.5–1ct、1–2ct		 0.5–1ct：低80–260 / 中260–700 / 高700–2000
1–2ct：低120–350 / 中350–900 / 高900–2800
>2ct：低180–500 / 中500–1300 / 高1300–4000 		Wikipedia
尖晶石
Spinel
（#4 提升客单）		 MgAl2O4 低：一般不涉及放射性担忧；按常规抽检 认知提升；灰紫/粉紫/蓝灰更亲民；适合日常轻奢 刻面：
0.3–1ct、1–2ct		 0.3–1ct：低200–700 / 中700–2000 / 高2000–5000
1–2ct：低300–1000 / 中1000–3000 / 高3000–8000
>2ct：低500–1500 / 中1500–4500 / 高4500–12000 		Wikipedia
成色分级口径：
• 低：颜色/光效弱、雾感重、切工一般、瑕疵明显（但仍可卖“性价比”）
• 中：颜色/光效明显、透明度较好、切工规整（适合主销）
• 高：颜色/光效强、通透、切工佳、瑕疵少（适合拉高客单与口碑）
